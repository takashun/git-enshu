# 岩手の魅力を伝えるサイト
岩手の魅力を伝えるサイトを作りました。

## ファイルについて

### charm*.html
岩手の魅力について説明したページです。

### style.css
各ページを装飾するファイルです。

### writer.html
ライター一覧です。こちらから各ライターの紹介ページへと遷移します。

## 留意点

ページの右側にリンク一覧を設けましたので、活用していただけたら幸いです。
